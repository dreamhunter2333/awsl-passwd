# 密码管理器 (WailsPassword)

使用 Wails 框架开发的跨平台密码管理工具。

## 功能特性

- 账号管理：添加、编辑、删除账号信息
- 密码存储：安全存储账号名、密码和备注信息
- 二步验证：支持 TOTP 二步验证
- 一键复制：支持复制账号名、密码、OTP验证码、备注到剪贴板
- 本地存储：数据存储在用户本地目录

## 构建和运行

### 开发模式

```bash
wails dev
```

### 构建

```bash
wails build
```

## 技术栈

- Go 1.23+
- Wails v2.10.1
- HTML/CSS/JavaScript
